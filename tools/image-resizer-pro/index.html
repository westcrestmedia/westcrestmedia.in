Questions

Frequently Asked

What image formats does the resizer support?
The Image Resizer Pro supports JPG, PNG, WEBP, GIF, BMP and AVIF. Upload any of these formats and export to your chosen output format in one step.
Can I resize multiple images at once?
Yes! Upload up to 50 images and resize them all with the same settings in one click. Download each image individually or grab them all as a single ZIP file.
Are my images uploaded to a server?
Never. All resizing happens entirely inside your browser using the Canvas API. Your images never leave your device, so everything stays completely private.
Can I lock the aspect ratio while resizing?
Yes. Aspect ratio lock is on by default — change the width and the height adjusts automatically, and vice versa. You can turn it off anytime to enter fully custom dimensions.
